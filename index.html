Search-backed 2025-2026 research notes

Important: Public sources are not fully consistent, and Infosys can change patterns by hiring channel. This app therefore uses a strict 75-question mock requested by you while timing sections using recent public reports where available.
  • PapersAdda reported an SP/general online assessment variant with Reasoning 15Q/25m, Mathematical 10Q/35m, Verbal 20Q/20m, Pseudocode 5Q/10m and Coding 2 problems/60m, no negative marking in that table.
  • Another PapersAdda 2026 SP page reported Phase 1 aptitude sections and Phase 2 coding with browser-locking, 2 coding problems and negative marking in Phase 1 for some 2024-25 drives.
  • FITA described 2024-style SP/DSE pattern updates including logical reasoning 15 questions/25 minutes, verbal ability 20 questions/20 minutes, and newly introduced pseudocode/puzzle-style sections.
  • PrepInsta and GeeksforGeeks interview experiences emphasize SP/Power Programmer coding practice with arrays, strings, dynamic programming, and medium-to-hard DSA style questions.
  • Official Infosys Certification guidelines mention web-proctored conduct such as not switching away from the test screen; this app mimics the feel without actually proctoring.
